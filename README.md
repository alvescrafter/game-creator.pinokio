# 🎮 Game Creator — Pinokio Launcher

A **Prompt Orchestrator** that turns module-based game design into a complete, playable HTML5 game generated by your AI of choice. Configure genre, mechanics, visuals, menus, and audio in the sidebar — the app assembles a detailed prompt and streams back a single-file, self-contained game.

This project is also a **Pinokio launcher** — the same `index.html` runs in Pinokio's built-in browser with no extra setup.

---

## 🚀 Run in Pinokio

1. Install [Pinokio](https://pinokio.co) desktop app.
2. **Install from URL**: paste the GitHub repository URL for this project.
3. Click **Start** — Game Creator opens in the Pinokio browser.
4. Click **🤖 Connect Your AI** in the top-right to add your API key.
5. Click **⚡ Generate Game** to create your first game.

> The app is a serverless web app — no server, no build step, no dependencies. The launcher just opens `index.html`.

## 🤖 Supported AI Providers

| Provider | API Key Required | Base URL |
|----------|-----------------|----------|
| OpenAI | Yes | `https://api.openai.com/v1` |
| Gemini | Yes | `https://generativelanguage.googleapis.com/v1beta/openai` |
| Claude | Yes | `https://api.anthropic.com/v1` |
| Ollama | No | `http://localhost:11434/v1` |
| LM Studio | No | `http://localhost:1234/v1` |

API keys are stored only in your browser's localStorage — never sent anywhere except the chosen provider.

## ✨ Features

- **5 configurable modules** — Core Identity, Mechanics, Visual, Game Menu, Audio (each independently toggleable)
- **10 genres** — Platformer, RPG, Puzzle, Shooter, Idle, Racing, Strategy, Roguelike, Simulation, Visual Novel
- **Click-to-toggle mechanic tags** — Double Jump, Inventory, Health Bar, Gravity, Permadeath, Leveling, Crafting, Stealth, Procedural Generation, Physics, Dialogue, Collectibles
- **Custom color palette** — Primary, Secondary, Background
- **8 art styles** — Pixel Art, Minimalist Vector, ASCII, Low-Poly 3D, Hand-drawn, Flat Design, Retro CRT, Neon Glow
- **Mandatory game structure** — every game ships with Title Screen, Pause Menu (ESC), and Game Over screen
- **Refine bar** — incrementally tweak the game without regenerating from scratch
- **Prompt preview** — see and copy the assembled prompt before generating
- **History** (up to 50 saved games) and **Templates** for reusable configurations
- **Download** finished games as a single `.html` file
- **Fullscreen** sandbox and **refresh** controls
- **Dark/light theme** toggle (follows system)
- **Keyboard shortcuts** — `Ctrl+Enter` generate, `Ctrl+B` sidebar, `Ctrl+H` history, `Ctrl+T` templates, `Ctrl+,` settings

## 🛠️ Programmatic Use

This is a single-page web app with no backend. There is no API to call programmatically from outside the browser — you interact with it via the UI. If you want to drive the prompt assembler from a script, open the page in a headless browser (Playwright, Puppeteer) and call the same functions the UI uses.

### Copy the assembled prompt from the console

```javascript
// Open index.html, then in DevTools console:
JSON.stringify(window.__getPrompt ? window.__getPrompt() : 'not exposed')
```

(The current build does not expose this globally — use the in-app **📋 Copy** button in the Prompt Preview tab instead.)

### Generate a game via fetch (from the browser)

```javascript
// Example: send the same prompt your browser would send.
// Replace the URL and key with your chosen provider's values.
const apiSettings = JSON.parse(localStorage.getItem('gameCreator.api.v1'));
const { systemPrompt, userPrompt } = /* call the UI's assemble function */;

const res = await fetch(`${apiSettings.baseUrl}/chat/completions`, {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${apiSettings.key}`
  },
  body: JSON.stringify({
    model: apiSettings.model,
    messages: [
      { role: 'system', content: systemPrompt },
      { role: 'user', content: userPrompt }
    ],
    temperature: apiSettings.temperature
  })
});
const data = await res.json();
console.log(data.choices[0].message.content);
```

### Curl example (OpenAI)

```bash
curl https://api.openai.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $OPENAI_API_KEY" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role":"system","content":"You are an expert Game Developer..."},
      {"role":"user","content":"Generate a platformer with double jump..."}
    ]
  }'
```

For the full system prompt template and the mandatory game structure rules, see `app.js` (search for `assemblePrompt`).
